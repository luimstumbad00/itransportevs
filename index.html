Nuestra app

MI transporte se basa en el desarrollo de una aplicación web y una aplicación para dispositivos Android. Estas aplicaciones tienen como objetivo facilitar la movilidad de los usuarios en la Red de Movilidad Integrada, haciendo que sea más comprensible y accesible tanto para aquellos que están empezando a utilizarla como para aquellos que la han estado utilizando durante años pero aún desconocen rutas más eficientes o formas más rápidas de llegar a sus destinos.





Funcionalidades



+ Planificador de rutas

+ Consulta de mapa y estaciones

+ Multiples opciones de visualizacion de ruta

+ Versión offline (limitada) para dispositivos móviles, además de la online móvil y web con funcionabilidades completas

+ Estimación de costos del trayecto de manera offline

+ Mini juegos de acceso de manera offline
